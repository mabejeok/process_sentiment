The aspect mining and topic modelling portions are in plp_project_aspect.ipynb

The output from LIME library using ten sample reviews are in the html files

To generate sentiment.csv that is used in the system code, please download the entire zip file, unzip, then go to https://drive.google.com/drive/folders/1SzV0lj7K_orHpwgD3fwRhvn4K7Sr6Zj3?usp=sharing, download the four model files and place them in sentiment_analysers folder. Then, run process_reviews.py using your favourite IDE or via command line using python process_reviews.py
